== Description ==

The GoCrypto Pay extension gives your online store a real-time competitive advantage, introducing a new and trending payment method to your customers. By enabling crypto payments you gain access to over 40 million new customers from all over the world.

You will be able to accept payments in over 50 different cryptocurrencies and swap between instant crypto settlements or zero volatility risk settlements in your local currency.

The crypto payment network is secure and complies with all relevant ISO standards and mandatory regulations.

GoCrypto Pay will make your online store stand out from the competition. Be among the first to accept crypto payments today.

= Web Payments API Support =

GoCrypto Pay includes [Web Payments API](https://www.w3.org/TR/payment-request/) support, which means customers can pay using payment details associated to their mobile devices, in browsers supporting the Web Payments API (Chrome for Android, amongst others).

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you donâ€™t need to leave your web browser. To do an automatic installation of the GoCrypto Pay plugin follow these steps:
1. Log in to WordPress admin.
2. Go to Plugins > Add New.
3. Search for the GoCrypto Pay plugin.
4. Click on Install Now, wait until the plugin is installed successfully and click on Activate.

= Setup =

To set up the GoCrypto Pay you need to have a GoCrypto account. If you do not have an account yet sign up > here. Follow the steps below to connect the plugin to your GoCrypto account:
1. After you have activated the GoCrypto Pay plugin, go to WooCommerce > Settings.
2. Click the Payments tab.
3. Enter the Host, OTP and Terminal ID that you received on your email address.
4. Click Save changes.
